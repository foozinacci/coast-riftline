# Sound Effects Setup

This game uses CC0 (public domain) sound effects from Kenney.nl.

## Download Instructions

1. Download these free asset packs from Kenney.nl:
   - **Sci-fi Sounds**: https://kenney.nl/assets/sci-fi-sounds
   - **Impact Sounds**: https://kenney.nl/assets/impact-sounds
   - **Interface Sounds**: https://kenney.nl/assets/interface-sounds

2. Extract and copy these files to `public/audio/`:

### Required Files:
```
public/audio/
├── laserSmall_000.ogg    (from Sci-fi Sounds - weapon fire)
├── laserLarge_000.ogg    (from Sci-fi Sounds - laser)
├── explosionCrunch_000.ogg (from Impact Sounds - explosion)
├── impactMetal_000.ogg   (from Impact Sounds - hit)
├── lowFrequency_explosion_000.ogg (from Impact Sounds - death)
├── click1.ogg            (from Interface Sounds - click)
├── rollover1.ogg         (from Interface Sounds - hover)
├── confirmation_001.ogg  (from Interface Sounds - confirm)
├── switch1.ogg           (from Interface Sounds - pickup)
├── footstep05.ogg        (from Impact Sounds - footstep)
└── powerUp1.ogg          (from Interface Sounds - respawn)
```
